# Etiquetas de producto

Para configurar que un producto de Logicommerce cargue un modelo 3D de threedium, hay que añadir 3 etiquetas del grupo de etiquetas THREEDIUM.

  • distID: Esta etiqueta es la unica opcional. Esta etiqueta indica que versioón de publicación queremos del modelo 3D de threedium. Si no se especifica, será la ultima generada automaticamente. Se puede conseguir accediendo al modelo3D de threedium.

  • solution3DID: Este valor es el id de threedium del modelo. Siempre es el mismo para cada modelo 3D.
  • solution3DName: Este valor es el nombre interno de threedium del modelo. Siempre es el mismo para cada modelo 3D.

# Mapa opción

Partes de la opción que se usan en el personalizador

  • Id interno: Esta compuesto por tres valores separados por ___ (tres "_").
    • El primer valor es el id del paso (para agrupar diferentes opciones en un mismo paso).
    • El segundo valor es un valor interno de threedium que variará segun la opción y del paso.
    • El tercer es identificativo, para nosotros identificar claramente de que opción se trata, tambien variará segun la opción y el paso.
  • Imagen: Aquí añadiremos el type, es un valor que indicarà como se tiene que comportar la opción dentro del step o entre steps. Los valores son predefinidos y estan listados en esta documentación. En este ejemplo es: SIMPLE_MATERIAL
  • Nombre: Si la opción forma parte de un step con mas de una opción, el nombre tiene dos valores separados por &&. El primer valor es el que sale en el menu de steps y el segundo el que sale en las pestañas dentro el step. En este ejemplo es Puntera && Piel



    Si tenemos un step con tres opciones, el primer valor tiene que ser identico en las 3.
  • Mensaje de Front office: Este contenido se mostrarà como descripción de la opción del step:



    Si se añade al final $$ y un numero, se puede modificar la grid de como se mostrarán los valores de opción. Por ejemplo poniendo Text lorem ipsum $$ 3, los valores se verán en grid de 3 y cuadrados:

# Mapa valor opción

Partes del valor de opción que se usan en el personalizador.

  • Id interno: Esta compuesto por de 2 a 3 valores separados por ___ (tres "_").
    • El primer es el valor de la opción, que dependerà del type de la opción y step.
    • El segundo es un valor que hacer referencia interna a Threedium tambien depende de cada step.
    • El tercer es un valor opcional que se usa en ciertas opciones y steps para restringir valores.
  • Imagen pequeña: Es la imagen que se muestra como valor para seleccionar en el personalizador.
  • Imagen grande: Es la imagen que se muestra en el resumen del personalizador.
  • Valor: El valor de opcion de Logicommerce, es el que se enviarà al comprar el producto final y el valor que se enseña en el personalizador.

# Horma y suela

El step del horma y suela tiene 2 opciones de Logicommerce agrupadas.

# Suela

Configuración de la opción en Logicommerce:

Id interno Type Nombre Mensaje front office
Soles___Soles___Type SOLE_TYPE Horma y suela && Suela Configura tu <b>suela</b>

Los valores de la opción deben seguir este ejemplo para su id interno: Sole_Tomir_270_normal___Soles Todos los valores seràn iguales cambiando la parte de Sole_Tomir_270_normal, este valor es el nombre de la parte de Threedium.

En el modelo de Threedium tendremos varios tipos de suela. De cada una hay combinaciones pero en esta opción solo nos interesa poner una de cada tipo. Por ejemplo en Threedium en el modelo tenemos estas suelas y sus variaciones, pero cogeremos estos dos valores:



De esta manera tendremos dos valores de opción en logicommerce con estos ids: Sole_Tomir_270_normal___Soles y Sole_Leather_270_normal___Soles

Restricciones
Los ids de los valores de esta opción restringen los valores de la opciones color de suela y color de canto. Cada suela tiene varias partes separadas por _: Sole, Tomir, 270 y normal, es importante el segundo parametro, el id de la suela. Con este id restringiremos los valores de las otras opciones.

# Color

Configuración de la opción en Logicommerce:

Id interno Type Nombre Mensaje front office
Soles___Soles___Color SOLE_COLOR Horma y suela && Color Selecciona tipo de piel y color de la <b>suela</b>

Los valores de la opción deben seguir este ejemplo para su id interno: Sole_bottom_Kakie___Soles___Tomir Tiene 3 partes, Sole_bottom_Kakie que es el color/material, el segundo que es un id interno de threedium y el tercer parametro Tomir es un id de suela.

Restricciones
Todos los valores de esta opción serán restringidos. Se mostraran los valores cuyo tercer parametro coincida con el id del tipo de suela seleccionado.

Por ejemplo, si en la opción tipo de suela seleccionamos Sole_Tomir_270_normal___Soles, en esta opción se mostrarán todos los valores que tengan como tercer parametro Tomir.

Configuración de la opción en Logicommerce:

Los valores de la opción deben seguir este ejemplo para su id interno: picado_0_270___Picado___Stormwelt Tiene 3 partes, picado_0_270 que es el valor de la parte de threedium, el segundo es un id interno de threedium que siempre serà igual, y el tercer parametro es opcionalStormwelt, que es un valor estatico.

El primer valor del id interno es una parte de threedium y pueden ser varias:


Por cada valor se pueden hacer combinaciones con Stormwelt. Así pues por ejemplo podemos tener picado_0_270___Picado y picado_0_270___Picado___Stormwelt.

Tambien podemos poner NONE como primer parametro, por ejemplo NONE___Picado como id de valor de opción "Sin vira picado".

# Opciones del resumen

Si en el campo imagen de la opción como "type" ponemos RESUME, esa opción automaticamente se mostrarà en el paso final del personalizador.

     

# Talla i puntera de metal

Adicionalmente para que la opción de talla se muestre y se trate como opcion diferente (La de talla) hay que cambiar el tipo (campo imagen) a RESUME i el tipo de opción a Talla.



Para la opción de puntera de metal (para que se una dentro de Talla) la configuración es la siguiente: Añadir como campo de imagen RESUME, de tipo de opción Otros y que la opción sea checkbox (opción de selección multiple).
